## Descripción

Stock Master es una aplicación para la gestión y control de inventario, desarrollada con **Java** y **Spring Boot** en el backend, y **Angular** en el frontend. Permite administrar productos, controlar el stock disponible, y visualizar información relevante del inventario en tiempo real.

## Tecnologías usadas

- **Backend:** Java, Spring Boot
- **Frontend:** Angular (TypeScript), HTML, CSS

## Características principales

- Gestión de productos: alta, baja y modificación.
- Control de stock en tiempo real.
- Interfaz intuitiva y responsiva.
- Historial de movimientos de inventario.
- Notificaciones de bajo stock.

## Instalación

### Backend (Java + Spring Boot)

1. Clona el repositorio:
   ```bash
   git clone https://github.com/INizetich/Stock-Master.git
   ```
2. Accede a la carpeta del backend y configura las variables de entorno (base de datos, puerto, etc).
3. Ejecuta la aplicación Spring Boot:
   ```bash
   ./mvnw spring-boot:run
   ```
   o
   ```bash
   mvn spring-boot:run
   ```

### Frontend (Angular)

1. Accede a la carpeta del frontend.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta la aplicación Angular:
   ```bash
   ng serve
   ```
4. Accede a [http://localhost:4200](http://localhost:4200) en tu navegador.

## Uso

- Ingresa productos nuevos y actualiza información existente.
- Visualiza el stock disponible y recibe alertas cuando el inventario esté bajo.
- Consulta el historial de movimientos para auditoría y trazabilidad.
